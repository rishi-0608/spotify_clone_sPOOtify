# 🎵 sPOOtify – Java Spotify Clone

A console-based Spotify clone developed as a case study project using Java and Object-Oriented Programming concepts.

---

## 🙌 Credits

Original project: https://github.com/duvrdx/sPOOtify

This project was used as a base for learning and further enhanced as part of a group case study.

---

## 🚀 Features

* Playlist management system
* Supports multiple content types:

  * 🎶 Music
  * 🎙️ Podcast
  * 📚 Audiobook
* Add, remove, and manage playlists
* Structured using OOP principles

---

## ✨ Enhancements & Contributions

The following improvements were implemented over the original project:

### ⏱️ Duration Handling (Sowmya)

* Converted duration from **seconds to minutes**
* Improved readability and consistency for playlists and podcasts

---

### 📂 Playlist Validation Logic (Rishvita)

* Ensured content can only be added to a playlist if it exists in the **main library**
* Prevents invalid entries and maintains data consistency

---

### 🧾 Content Display Fix

* Fixed issue where **authors and interpreters were not displayed** for podcasts and audiobooks
* Updated output logic to correctly show all contributors

---

### 🖥️ Output Formatting

* Used `toString()` method overriding for clean and structured display of content details
